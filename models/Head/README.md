## Head STL Files

| Part | Role | Material | File |
|---|---|---|---|
| Head1 | Main head structure. Includes mounting holes for the head electronics board and a cutout for the ribbon cable. | PETG | [Head1.stl](./Head1.stl) |
| Head2 | Head cover. Sits on top of the head, includes mounting holes for the camera ring light and cutouts for the screen, microphone, and speaker. | PETG | [Head2.stl](./Head2.stl) |
| Head3 | Base of the head mounting mechanism. Screws into Head2 and holds the dual-servo pan-tilt support and ToF sensor mount. | ASA | [Head3.stl](./Head3.stl) |
| Head4 | First part of the head mechanism that allows left-right rotation. | ASA | [Head4.stl](./Head4.stl) |
| Head5 | Second part of the head mechanism that provides up-down movement and includes the mount for the ToF sensor. | ASA | [Head5.stl](./Head5.stl) |
| Head6 | Support cylinder for Head4, added to reinforce the head rotation mechanism. | ASA | [Head6.stl](./Head6.stl) |
